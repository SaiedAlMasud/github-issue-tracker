# github-issue-tracker

1. var is the old way to declare variables and it's function-scoped.
 let is the new way for variables that it is block-scoped,
 const is for things that should never change.

2. Spread operator is like taking items out of an array or object and spreading them out.

3. map(): loops through an array and returns a new array of same length, 
filter(): returns a new array with only items that pass a condition,  
forEach(): just loops through but returns nothing - it's like a basic for loop. I have to write statement to return something.

4. Arrow function is a shorter way to write functions using (=>) symbol

5. Template literals are strings with backticks `` that let you add html code with javascript variable name using ${}
